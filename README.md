# Forge Quote Tool

## Project Purpose
The Forge Quote Tool is designed to help users quickly generate quotes for various applications based on predefined templates. It aims to streamline the quoting process, allowing for faster turnaround times and improved accuracy.

## Features
- **Template Management:** Easily create and manage quote templates.
- **Customization:** Personalize quotes with user-specific information.
- **Export Options:** Export quotes in different formats such as PDF or Word.
- **User-Friendly Interface:** A simple and intuitive UI that makes generating quotes easy.

## Setup Instructions
1. **Clone the repository:**  
   ```bash  
   git clone https://github.com/soumyaghosh221999-source/forge-quote-tool.git  
   cd forge-quote-tool  
   ```  
2. **Install dependencies:**  
   ```bash  
   npm install  
   ```  
3. **Run the application:**  
   ```bash  
   npm start  
   ```  
4. **Access the application:**  
   Open your web browser and go to `http://localhost:3000`.
